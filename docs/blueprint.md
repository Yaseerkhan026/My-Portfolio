# **App Name**: DevFolio Pro

## Core Features:

- Personal Bio & Introduction: A dynamic section to display Yaseer Khan's name, designation (e.g., GDE - Firebase), and an introductory statement, easily updatable via component props.
- Experiences & Achievements Display: Showcase Yaseer Khan's academic background (JIT), past experiences (IANT), and notable achievements (NPTEL score) through prop-driven UI components.
- Skills & Technologies Showcase: A section to list all known technologies (HTML, CSS, Java, JavaScript, React, Python, Firebase), presented using modular, prop-driven components.
- Contact Information Block: Display contact details (email, phone, LinkedIn) with appropriate, visually distinct icons and clickable links.
- Responsive Layout Integration: Ensure the portfolio adapts seamlessly to various device sizes (mobile, tablet, desktop) using Bootstrap's responsive grid and utility classes.
- AI Bio Refinement Tool: An AI-powered tool to help Yaseer Khan generate or refine concise, impactful drafts for the 'About Me' section or project summaries, leveraging generative AI to improve readability and professionalism.

## Style Guidelines:

- Light theme based on a violet palette. Primary brand color: a soft lavender (#9C7CCD). Background: a very light, desaturated lavender white (#F5F3FA) to provide a clean canvas. Accent color: a vibrant blue-violet (#423DE6) for interactive elements and highlights to provide a clear contrast.
- Headline font: 'Poppins' (geometric sans-serif) for a modern and precise aesthetic. Body font: 'PT Sans' (humanist sans-serif) for excellent readability and a touch of warmth. Code font: 'Source Code Pro' (monospace sans-serif) for displaying any technical code snippets or terminal outputs.
- Use distinct, clear vector icons from a consistent library (e.g., Font Awesome or Lucide Icons) for contact details (email, phone, LinkedIn). Additionally, leverage simple icons to categorize experiences or highlight skills visually.
- Implement a clean, spacious, and modular layout using Bootstrap's grid system. Content sections should be well-defined with clear visual separation. Key information should be easy to locate, with a focus on a professional, polished appearance across all devices.
- Incorporate subtle hover effects on interactive elements like buttons and links to enhance user feedback. Use gentle, swift fade-in or slide-up transitions for content sections as they enter the viewport for a polished user experience.
